# GitHub Template Features

This section documents the GitHub template features included in this Spring Boot Multi-Module Project Template.

## Overview

This template includes a comprehensive set of GitHub features to improve project management, collaboration, and code quality:

- Issue templates for structured bug reports and feature requests
- GitHub Actions workflows for CI/CD
- Community health files for project governance
- Issue labels for better organization
